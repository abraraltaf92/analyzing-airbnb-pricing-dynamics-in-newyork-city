# Airbnb Pricing Analysis in New York City

## Overview
This project analyzes Airbnb pricing dynamics in New York City using data mining techniques.
The goal is to understand pricing patterns, identify natural market segments, and predict
listing prices using regression models.

## Dataset
- Source: Inside Airbnb (New York City)
- File used: `listings.csv`
- Approximately 20,900 listings after preprocessing

## Methods
- Exploratory Data Analysis (EDA)
- K-Means Clustering
- Linear Regression
- Random Forest Regression
- Feature Importance Analysis

## Key Results
- Identified 5 meaningful clusters of Airbnb listings
- Random Forest significantly outperformed Linear Regression
- Key price drivers include room type, location, minimum nights, and availability

## Repository Structure
- `data/` – raw and processed datasets
- `notebooks/` – Jupyter notebook with full analysis
- `figures/` – saved plots and visualizations
- `report/` – LaTeX project report
- `presentation/` – presentation slides
